# Data Directory

## Structure

```
data/
├── raw/           ← Place downloaded Kaggle CSVs here (git-ignored)
├── processed/     ← Generated by preprocess.py (git-ignored)
└── results/       ← Generated by simulation.py (git-ignored)
```

## Downloading the Datasets

Three publicly available Kaggle datasets are used as proxy data for Keele University campus parking.  
You will need a free Kaggle account to download them.

### 1. PKLot Parking Occupancy Dataset

- **URL:** https://www.kaggle.com/datasets/blanderbuss/parking-lot-dataset
- **File to save as:** `data/raw/pklot_occupancy.csv`
- **Records:** ~695,899
- **Timestamp format:** `DD/MM/YYYY HH:MM`

### 2. CNRPark+EXT Occupancy Dataset

- **URL:** https://www.kaggle.com/datasets/ammarnassanalhajali/cnrpark-ext
- **File to save as:** `data/raw/cnrpark_occupancy.csv`
- **Records:** ~144,965
- **Timestamp format:** ISO 8601 (`YYYY-MM-DDTHH:MM`)

### 3. Smart Parking Sensor Dataset

- **URL:** https://www.kaggle.com/datasets/mypapit/smart-parking-system
- **File to save as:** `data/raw/smart_parking.csv`
- **Records:** ~49,120
- **Timestamp format:** Unix epoch (seconds)

## After Downloading

Run the preprocessing pipeline:

```bash
python src/preprocess.py
```

This will produce `data/processed/unified_parking.csv` (~150,000 records) with:

| Column | Description |
|---|---|
| `timestamp` | 15-minute interval datetime |
| `bay_id` | Integer bay identifier (0–1999, calibrated to Keele ~2,000 bays) |
| `occupancy` | Binary (0 = free, 1 = occupied) |
| `source` | Original dataset name (PKLot / CNRPark / SmartParking) |

## Notes

- All datasets are anonymised and contain no personal data.
- Ethical approval for use of these public proxy datasets is covered under Ulster University Category Z.
- The preprocessing pipeline filters records to weekdays, 08:00–18:00, matching Keele's enforcement hours.
